# kisskittens.xyz

Integrated static prototype for `kisskittens.xyz`.

The homepage and primary brand remain Kiss Kittens. The massage direction is mounted under `spa/` as a sub-section:

- `/` / `index.html` — Kiss Kittens homepage.
- `/spa/` / `spa/index.html` — Velour / Kiss Kittens Spa.
- `/spa/Rituals.html`, `/spa/Therapists.html`, `/spa/Booking.html` — massage section pages.

The repository root is the deployable web root. Upload or sync these files directly into the hosting document root, for example `public_html`.

This is still a static prototype. Scheduling, applications, and admin editing require a backend, CMS, or connection to the existing forms/schedule system.
