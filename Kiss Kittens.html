My Recommendation

Horizontal lockup on black, "KISS" in pink, the facing-in monogram to the left. Keep the lip mark as a secondary stamp only — for social, stickers, a wax seal on invites — never in the core logo, so the brand stays elevated, not novelty. Same type system (Archivo Expanded), same palette.
